# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately and do not open a public issue.

- Email: contact@utildex.org
- Subject: `Security report - Utildex`

Please include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Impact assessment
- Suggested remediation (if known)

## Response Expectations

- Initial acknowledgement target: within 7 days
- Triage/update target: within 2 weeks
- Fix timing: depends on severity and complexity

We will coordinate responsible disclosure and publish a fix note once a patch is available.
